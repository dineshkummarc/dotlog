DotLog is minimalist logging library for .NET

There are a number of great logging options for .NET applications, but sometimes all you need is a quick and easy way to add basic log file support to a simple or prototype application without a lot of extra baggage or configuration.

DotLog is a lightweight solution that makes it easy to add to file logging to your project. It provides a Common.Logging adapter, so as your requirements grow, you can easily switch to a more sophisticated logging framework.
